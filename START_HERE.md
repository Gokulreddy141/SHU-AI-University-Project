## 🤖 What This Project Does

This is an AI-powered online examination proctoring system with:

- **29+ AI Detection Systems** running in real-time
- **Computer Vision**: Face detection, gaze tracking, hand gestures, object detection
- **Audio Analysis**: Voice activity, TTS detection, lip-sync verification
- **Behavioral Analytics**: Keystroke dynamics, mouse patterns, response time
- **System Monitoring**: Virtual camera, screen recording, multi-tab detection

## 🎬 5-Minute Demo Script

1. **Start**: Open AI Diagnostics page
2. **Face Detection**: Show face counter (1 → 2 when someone enters)
3. **Gaze Tracking**: Look away → indicator triggers
4. **Phone Detection**: Hold phone → object detection triggers
5. **Voice Activity**: Speak → voice indicator activates
6. **TTS Detection**: Play robotic audio → spoofing detected
7. **Lip-Sync**: Play audio without moving lips → anomaly detected
8. **Integrity Score**: Show how violations affect score
